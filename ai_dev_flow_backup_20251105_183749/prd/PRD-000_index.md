# PRD Index (Product Requirements)

Purpose

- Central index for Product Requirements Documents (PRDs).
- Tracks allocation and sequencing for `PRD-NNN_{descriptive}.md` descriptors.

Allocation Rules

- Numbering: allocate sequentially, starting at `001`.
- Ensure each descriptor links to upstream business context (BRDs) and downstream specifications per repository standards.
- Each PRD should link to downstream EARS, REQ, and SYS documents.
