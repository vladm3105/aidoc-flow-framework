‚öôÔ∏è Threshold Management

Threshold Definition Strategy: Thresholds are defined in source documents (BRD/PRD/ADR) and referenced via @threshold: tags in downstream artifacts‚Äîeliminating separate registry documents while maintaining full traceability.

Threshold Categories

Category Examples Defined In
quota User limits, resource caps, tier allowances BRD, PRD
rate API rate limits, request throttling, burst limits BRD, PRD
perf P95/P99 latency, throughput targets, SLA metrics ADR
timeout API timeouts, session expiry, cache TTL ADR
circuit Failure counts, reset timeouts, pool sizes ADR
alert Warning thresholds, critical limits, escalation triggers ADR

Tag Format

@threshold: {DOC_TYPE}.{DOC_NUM}.{threshold_key}

Examples:
@threshold: PRD.01.quota.user.daily
@threshold: PRD.01.rate.api.standard
@threshold: ADR.15.circuit.failure.count
@threshold: ADR.15.perf.api.p95
